Span is a true inline element. No height, no width. The height and width of the text. Ignores height and width.

Block elements goes up-down or top-down. Height and width properties recognized. Defeault width is length of the browser window.

Inline-block is a hybrid. Half inline, half block. Recognize height and width properties. No new line break.
