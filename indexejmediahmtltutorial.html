  • Attributes gives more information about an element. It also gives more functionality to an element. Attributes are in the openeing tag.

Meta Element

  • Always put inside the head element. They're not visible to the web user.
  • Tell search engines your webpage using meta attributes.
  • Use meta tag. No closing tag. End with a "/>"
  • meta name="description" content="This is a web page about fish. We cover all the fish in the world." /> provides description to the search engine. Max length 155 characters. If there's no description, the browser does it for you.
  • meta name="keywords" content="fish, ocean, aquarium" /> provides keywords a web user might search. Most search engines including Google doesn't use keywords. Foreign search engines may use them.
  • meta http-equiv="author" content="Raymond Mar" /> author of the website
  • meta http-equiv="content-language" content="en-us" /> language of the website
